# Workflow: SDD Apply

Uses the `sdd-apply` skill to implement tasks from `tasks.md`.

## Usage

```
/sdd-apply [change-name]
```

## Steps

1. Read `.agents/skills/sdd-apply.md` for the full skill instructions.
2. Read context: `tasks.md`, `design.md`, `specs/`, `proposal.md`.
3. Work through unchecked tasks one at a time, marking `[x]` when done.
4. Announce completion and prompt the user to run `/sdd-verify`.
